行列表中列的删除和排序

Date 2013-05-22 17时53分32秒 By Elvin.Cracker Category Python. Tags Python List

如果你有一个包含列表（行）的列表，现在你想获得另外一个列表，该列表包含了同样的行，但是其中一些列被删除或者被排序了，那应该如何处理呢？

假设你有如下列表：

list_of_rows = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

现在你需要一个同样的列表，但是第二列被删除了，并且第一列与第三列调换了顺序。你可以使用列表推导的方式来实现，如下：

list_of_news = [[row[2], row[0]] for row in list_of_rows]

你可以在列表推导的内部使用一个嵌套的辅助序列（列表或元组），这样也许看起来更优雅一些，如下：

list_of_news = [[row[i] for i in (2, 0)] for row in list_of_rows]
